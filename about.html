Stream Monitor

Automatically opens Twitch streams when your favorite streamers go live, and closes tabs when they raid.

Welcome! This page opens once after a fresh install or update. It won't appear again automatically. You can always find it in the About menu.

What's included

  • Desktop app that monitors Twitch and opens streams automatically
  • Firefox extension that detects raids and closes tabs
  • Optional auto-mute for opened streams
  • System tray controls and automatic update checking

Firefox Extension

Install the companion Firefox extension to automatically close tabs when a streamer raids. Click the extension icon to toggle auto-mute and view debug logs.
